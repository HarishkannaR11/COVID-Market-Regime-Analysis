### Panel Structure

### What is Cumulative Activation?
**Definition**: Running total of active days for each system from the start of the period.

**Calculation**:
```
Cumulative_Activation[date] = Sum of all activation days from start to current date
```

### Systems Tracked (5 brain analogs)
1. **Value** (orange) - Reward/valuation processing
2. **Risk** (red) - Risk assessment and aversion
3. **Sentiment** (green) - Trend following and directional bias
4. **Insula** (purple) - Uncertainty and interoceptive awareness
5. **Control** (blue) - Executive control and inhibition

### Key Insight from This Graph
**The SLOPE of each line represents the ACTIVATION RATE (frequency)**
- Steeper slope = Higher activation frequency
- Flatter slope = Lower activation frequency
- Changes in slope = Changes in activation patterns

## APPENDIX: TECHNICAL DETAILS

### A. Graph Reading Guide

**How to Read Cumulative Plots**:
1. **Y-axis**: Total accumulated activations since period start
2. **Slope**: Activation frequency (activations per day)
3. **Steeper = More frequent**, Flatter = Less frequent
4. **Final value**: Total activations over entire period
5. **Relative position**: Ranking of frequency

### B. Calculation Method

```
For each system:
  cumulative[day_1] = activation[day_1]
  cumulative[day_t] = cumulative[day_t-1] + activation[day_t]
  
Activation frequency = cumulative[final_day] / total_days
Slope ratio = frequency_post / frequency_pre
```
